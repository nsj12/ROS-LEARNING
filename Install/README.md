# 本文件夹为ros安装的脚本

## ros1.sh

1. 为ros1安装脚本

2. 安装版本为：noetic

3. **注意**：不同的ubuntu版本对应的ros版本不一样

   ```shell
   ROS Noetic Ninjemys #ubuntu20.04
   ROS Melodic Morenia #ubuntu18.04
   ROS Kinetic Kame    #ubuntu16.04
   ```

4. 测试

   ```shell
   source /opt/ros/noetic/setup.bash #对应你下载的版本
   ```

   

## ros2.sh

1. 为ros2安装脚本

2. 安装版本为：galactic

3. 想安装不同版本的ros2请修改

4. 测试

   ```shell
   source /opt/ros/galactic/setup.bash
   ```
